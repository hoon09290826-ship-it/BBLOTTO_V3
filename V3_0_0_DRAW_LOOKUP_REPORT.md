# BBLOTTO PRO V3.0.0 STABLE - 회차별 당첨번호 조회 기능 추가

## 작업 내용
- 1회차부터 추첨 완료 최신 회차까지 회차 번호로 당첨번호를 조회하는 API 추가
- DB에 저장된 회차는 즉시 조회
- DB에 없는 회차는 동행복권 공개 조회를 시도하고 성공 시 DB에 자동 저장
- 아직 추첨 전 회차는 저장/조회하지 않고 안내 메시지 표시
- 프론트 당첨번호 확인 화면에 회차별 조회 입력칸 추가
- 조회한 당첨번호를 당첨확인 입력칸에 바로 적용하는 버튼 추가

## 추가/수정 파일
- backend/app.py
- frontend/index.html
- frontend/app.js
- V3_0_0_DRAW_LOOKUP_REPORT.md

## 추가 API
- GET /api/draws/search?round_no=회차

## 사용 방법
1. 당첨번호 확인 화면으로 이동
2. 회차별 당첨번호 조회 영역에서 회차 입력
3. 회차 조회 클릭
4. 필요한 경우 당첨확인에 적용 클릭

## 주의사항
- 인터넷 연결 또는 동행복권 조회 차단이 있으면 DB에 없는 과거 회차 자동 조회가 실패할 수 있습니다.
- 이미 DB에 저장된 회차는 인터넷 없이도 조회됩니다.
