# V3.0.0 등록일 수정 최종 수정 보고

## 수정 범위
- 등록일(created_at): 모든 관리자 수정 가능
- 계약기간(contract_months): 모든 관리자 수정 가능
- 계약만료일(contract_end_at): 등록일 + 계약기간 기준 서버 자동 계산
- 등록 관리자(created_by): 대표관리자만 변경 가능
- 일반관리자가 created_by 값을 보내도 서버에서 무시
