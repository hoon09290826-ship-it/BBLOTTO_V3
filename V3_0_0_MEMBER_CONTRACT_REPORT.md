# BBLOTTO PRO V3.0.0 - 회원관리 계약/등록관리자 개선

## 반영 내용
- 회원 등록/수정 화면에 `등록 관리자` 선택 항목 추가
- 대표관리자만 회원의 등록 관리자를 변경 가능하도록 제한
- 회원 등록일 수정 항목 추가
- 1년 계약 기준의 계약만료일 항목 추가
- 기존 회원 테이블에 `contract_end_at` 컬럼 자동 추가
- 신규 회원 등록 시 계약만료일을 입력하지 않으면 등록일 기준 1년 뒤로 자동 설정
- 회원 목록에 등록 관리자 / 등록일 / 계약만료일 표시
- 회원 CSV 내보내기에 계약만료일과 등록관리자 포함

## 검증
- `backend/app.py` Python 문법 검사 통과
- `scripts/verify_release.py` 통과
- 임시 DB 기준 `members.contract_end_at` 자동 생성 확인

## 주의
- 일반 관리자는 기존 권한 범위대로 본인이 접근 가능한 회원 정보만 수정할 수 있습니다.
- 등록 관리자 변경, 등록일 수정, 계약만료일 수정은 대표관리자 권한에서만 적용됩니다.
