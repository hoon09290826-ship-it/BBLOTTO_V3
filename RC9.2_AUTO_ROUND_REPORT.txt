BBLOTTO PRO V50 RC9.2 - AUTO ROUND ANALYSIS

변경 사항
1. AI 분석 목표 회차를 1231 고정값에서 한국시간 기준 최신 추첨 완료 회차로 변경
2. 매주 토요일 20:35 이후 새 회차가 분석 대상에 자동 포함
3. DB에 더 최신 회차가 저장되어 있으면 해당 회차까지 자동 분석
4. 관리자 AI 엔진 화면의 분석범위/저장회차/전체여부를 동적 회차로 표시
5. 전체 회차 동기화 버튼도 고정 회차 없이 현재 완료 회차까지 자동 동기화
6. 추천번호 생성 시 캐시가 최신 DB 회차와 다르면 자동 재분석
7. 서비스워커 캐시 버전 갱신

동작 예시
- 현재 완료 회차가 1232회이면 분석범위 1~1232
- 1233회 추첨 완료 후 자동 업데이트로 당첨번호가 저장되면 분석범위 1~1233
- 누락 회차가 있으면 관리자 AI 엔진 화면에 누락으로 표시되며 전체 동기화/분석 버튼으로 보강
